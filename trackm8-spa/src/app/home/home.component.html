How does it work?

  1. Get the data from the API using the APIService I created (I'm getting data from a mock API that I hosted, this project is using a proxy as I was getting errors trying to get the data)
  2. Once the data has been received and stored, the loading screen dissapears and the table is populated with data
  3. The table data is organised using the a pipe that i found online (https://stackblitz.com/edit/angular-uzzqda?file=app)
  4. When the delete button is clicked on an item, that item is selected in the array and then deleted (spliced)
  5. When the add button is clicked, that user enters the data they want, once it's submitted, validation checks are carried out, once its validated it gets pushed into the array
